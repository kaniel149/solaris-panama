# Solaris Panama — לידים לגגות סולאריים: פדאסי ↔ קניאס

## דרישות
- גגות מעל 400 מ"ר
- חיבור חשמל קיים
- אזור: פדאסי (Pedasí) עד קניאס (Cañas), Los Santos Province, Panama

## רגולציה — מה שצריך לדעת
- **Net Metering**: ASEP מאפשר עד 5% מסך האנרגיה, 16% מהביקוש השיא
- **Law 417 (2024)**: פטור ממס + פטור מכס על ציוד סולארי
- **תקרת מערכת**: עד 500 kW per installation (יש דיון להסיר)
- **170+ MW מותקנים** כבר בפנמה, 6,000+ users
- **קרינה ממוצעת**: 5-6 kWh/m²/day — מצוין לסולארי

## Value Proposition לבעלי נכסים
1. **Law 417**: פטור ממס + מכס = החזר ROI מהיר יותר
2. **חשבון חשמל גבוה** (AC + מקררים + תאורה) = חיסכון משמעותי
3. **Net Metering**: עודפים חוזרים לרשת = קרדיט
4. **Green branding**: תיירים eco-conscious מעדיפים מלון ירוק
5. **אנרגיה יציבה**: פנמה סובלת מהפסקות חשמל באזורים כפריים

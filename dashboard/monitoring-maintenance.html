DIAGNOSTICS

Troubleshooting Guide

Common issues and resolution procedures for Panama solar installations

Low Production Output HIGH

System producing less than expected for weather conditions.

Common causes: soiling, shading from new construction/vegetation, degraded panel, failed optimizer
→ Clean panels, trim vegetation, check per-panel data, replace failed components

Inverter Fault HIGH

Inverter displaying error code or not producing AC output.

Common causes: grid voltage out of range, overtemperature, isolation fault, internal component failure
→ Check error code in manual, verify grid voltage, ensure ventilation, cycle power, contact manufacturer

Hotspots Detected MEDIUM

IR scan reveals abnormally hot cells or connections.

Common causes: micro-cracks, poor soldering, bypass diode failure, partial shading
→ Identify affected panel via IR, check bypass diodes, replace panel if warranted

Shading Issues MEDIUM

New shading sources affecting system after installation.

Common causes: tree growth, new construction nearby, antenna/satellite dish installed
→ Trim vegetation, re-analyze shading profile, consider optimizer upgrade if needed

String Failure HIGH

One or more strings not producing power or showing zero voltage.

Common causes: blown fuse, loose MC4 connector, damaged cable, failed optimizer in series
→ Check fuses, inspect all MC4 connections, trace cable for damage, test each panel individually

Communication Loss LOW

Monitoring platform shows system offline or no data transmission.

Common causes: WiFi router issue, inverter WiFi module reset, internet outage, firewall blocking
→ Restart router, re-pair inverter WiFi, verify internet, check firewall rules

ALERT PROTOCOLS

Alert System

What triggers alerts, how they're classified, and response time commitments

🔴

Inverter Offline

Inverter stops producing AC power. Zero output during daylight hours. System completely down.

Response: 4 hours (Premium) / 24 hours (Standard)
🔴

Grid Fault Detected

Grid voltage or frequency out of acceptable range. Inverter anti-islanding triggered. System shuts down for safety.

Response: Monitor — usually auto-recovers
🟡

Production Drop >15%

System producing 15%+ less than expected based on irradiance data. May indicate soiling, shading, or component degradation.

Response: 48 hours investigation
🟡

String Underperformance

One string producing significantly less than others. Indicates potential panel, optimizer, or wiring issue.

Response: Next scheduled visit
🟢

Communication Loss

Monitoring data not received for 2+ hours during daylight. System may still be producing normally.

Response: Remote troubleshoot within 24h
🟢

Firmware Update Available

New inverter firmware released by manufacturer. Schedule update during low-production hours.

Response: Schedule within 2 weeks

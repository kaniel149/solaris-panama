System Commissioning Checklist

⚡

Technical Commissioning Tests

All tests must pass before client sign-off

  • IV Curve Test — verify each string's voltage and current match specificationsUse Seaward or Fluke IV curve tracer. Document results per string.
  • Insulation Resistance Test — minimum 1MΩ between conductors and groundTest DC cables, AC cables, and grounding conductors separately.
  • Grounding Verification — all metal components properly bonded to groundMeasure ground rod resistance: must be ≤25Ω per NEC. Use Megger ground tester.
  • Production Verification — system producing within 5% of design estimateMonitor for 2-3 sunny hours. Compare kWh to PVsyst simulation. Screenshot inverter app.
  • Visual Inspection — panels clean, wiring neat, labels applied, no debrisTake completion photos from all angles. Include close-ups of junction boxes and labels.
  • Monitoring Setup — FusionSolar/SolarEdge account created and linked to clientShare login credentials. Verify real-time data appearing in app.

ENSA Meter Activation

Bidirectional meter installation process with ENSA (2-4 weeks)

1. Submit Application

File interconnection request with ENSA. Include single-line diagram, inverter specs, and Ley 417 certification. Submit at ENSA office in Chitré or Santiago.

Day 1

2. ENSA Inspection

ENSA technician visits to verify installation meets interconnection standards. Must match submitted diagrams. Coordinate access with client.

Week 1-2

3. Bidirectional Meter Install

ENSA replaces the existing meter with a bidirectional net meter. No cost to client (covered by Ley 417). System can now export excess energy.

Week 2-4

4. Net Metering Active

Confirm credits appearing on client's next ENSA bill. Verify kWh exported vs. consumed are tracked correctly. Celebrate!

When to Call

Call us for: system producing 0 for 2+ days, inverter error codes, physical damage from storm. DON'T call for: cloudy day dips, night-time zero production.

Maintenance Schedule

Recommended maintenance intervals for optimal system performance

QUARTERLY

Panel Cleaning

Rinse panels with water early morning or late afternoon. No abrasive chemicals. In dry season (Jan-Apr), increase to monthly. Dust, bird droppings, and pollen reduce output 5-15%.

ANNUAL

Full Inspection

Check all connections, torque mounting bolts, inspect cables for UV damage, clean junction boxes, verify grounding, test inverter performance, compare annual kWh to design estimate.

5-YEAR

Inverter Check

Deep inverter diagnostic: fan operation, capacitor health, firmware update, DC/AC conversion efficiency test. Consider extended warranty purchase before year 10. Budget $200-400 for inspection.

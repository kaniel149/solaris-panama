Installation Execution Ejecución de Instalación ביצוע התקנה

Day-by-day execution plan for a flawless solar installation in Panama. Plan de ejecución día a día para una instalación solar impecable en Panamá. תוכנית ביצוע יום-יום להתקנה סולארית ללא רבב בפנמה.

Day-by-Day Installation Schedule Cronograma de Instalación Día a Día לוח זמנים יום-יום להתקנה

1
Mounting Structure & Roof Prep
7:00 AM – 4:00 PM | Team: 4 installers + 1 supervisor

Morning (7:00 – 12:00)

  • Site arrival, safety briefing, PPE verification (hard hats, harnesses, gloves)
  • Material staging: unload mounting rails, clamps, roof hooks from truck
  • Roof survey confirmation: verify structural attachment points vs. design
  • Mark rail positions on roof using chalk lines per engineering layout
  • Install roof hooks or L-brackets with proper sealant (Sikaflex for metal roofs)
  • First row of rails installed and leveled

Afternoon (12:30 – 16:00)

  • Complete all rail installation — verify level across entire array
  • Install mid-clamps and end-clamps (finger-tight, final torque after panels)
  • Route DC cable conduit pathways from array to inverter location
  • Install string junction boxes if required by design
  • Clean up, secure loose materials, photo documentation of Day 1 progress

Documentation Required

  • Photo: Roof before installation (baseline)
  • Photo: Each roof penetration with sealant applied
  • Photo: Complete rail layout with level verification
  • Photo: Team on site with PPE (safety compliance)
2
Panel Installation & DC Wiring
7:00 AM – 4:00 PM | Team: 4 installers + 1 electrician

Morning (7:00 – 12:00)

  • Panel transport to roof using panel lifter or human chain (never drag)
  • Install first string of panels — verify alignment with first 2 panels before continuing
  • MC4 connections as each panel is placed (click-lock verification)
  • Torque mid-clamps to spec (8-10 Nm for most K2 systems)
  • Continue panel installation across all strings

Afternoon (12:30 – 16:00)

  • Complete all panel installation — verify end clamp torque
  • DC string wiring: route PV cables through conduit to inverter location
  • Label each string at both ends (String 1+, String 1-, etc.)
  • Measure open circuit voltage (Voc) on each string — record and compare to design
  • Insulation resistance test on DC strings (>1 MΩ per string)
  • String Voc verification: should match N × panel Voc (±5%)
  • Photo documentation of completed array and all string measurements

Documentation Required

  • Photo: First panel placed showing alignment
  • Photo: Complete array from ground level
  • Photo: MC4 connections detail
  • Photo: Voc measurement readings per string
  • Log: String voltage readings table
3
Inverter, AC Connection & Commissioning
7:00 AM – 5:00 PM | Team: 2 installers + 1 electrician + 1 supervisor

Morning (7:00 – 12:00)

  • Mount inverter on wall bracket (shaded location, accessible for maintenance)
  • Connect DC strings to inverter MPPT inputs per wiring diagram
  • Install DC disconnect switch between array and inverter
  • Install AC disconnect between inverter and main panel
  • AC wiring from inverter to dedicated breaker in main electrical panel
  • Install Type II surge protection device (SPD) on AC side

Afternoon (12:30 – 15:00)

  • Ground fault and polarity verification on all DC connections
  • AC connection verification: voltage, phase, frequency
  • Close DC disconnects — inverter should detect PV voltage
  • Power on inverter — follow manufacturer commissioning sequence
  • Configure WiFi/dongle for FusionSolar monitoring platform
  • Set grid parameters per ASEP requirements (voltage limits, frequency, anti-islanding)
  • Close AC disconnect — system begins generating

Late Afternoon (15:00 – 17:00)

  • Monitor first hour of production — verify all strings producing equally
  • Install monitoring CT sensors on main feed if net metering
  • System performance snapshot: current power, daily yield estimate
  • Client walkthrough: show inverter display, app access, emergency shutoff
  • Final cleanup: remove all packaging, zip-tie loose cables, clean work area
  • Sign installation completion certificate with client

Documentation Required

  • Photo: Inverter mounted with all connections
  • Photo: Electrical panel with new solar breaker labeled
  • Photo: Monitoring app showing live production
  • Photo: Client receiving system walkthrough
  • Document: Signed completion certificate

Inversor, Conexión AC y Puesta en Marcha
7:00 AM – 5:00 PM | Equipo: 2 instaladores + 1 electricista + 1 supervisor

Mañana (7:00 – 12:00)

  • Montar inversor en soporte de pared (ubicación sombreada)
  • Conectar strings DC a entradas MPPT según diagrama
  • Instalar desconector DC entre arreglo e inversor
  • Instalar desconector AC entre inversor y panel principal
  • Cableado AC desde inversor a breaker dedicado en panel eléctrico
  • Instalar protección contra sobretensiones (SPD) Tipo II en lado AC

Tarde (12:30 – 17:00)

  • Verificación de falla a tierra y polaridad en todas las conexiones DC
  • Verificación de conexión AC: voltaje, fase, frecuencia

  • Encender inversor — seguir secuencia de comisionamiento del fabricante
  • Configurar WiFi/dongle para plataforma FusionSolar
  • Configurar parámetros de red según requisitos ASEP
  • Cerrar desconector AC — el sistema comienza a generar
  • Monitorear primera hora de producción
  • Recorrido con el cliente: mostrar display, app, apagado de emergencia
  • Limpieza final, firmar certificado de completación

Safety Briefing Checklist Lista de Verificación — Charla de Seguridad רשימת תדריך בטיחות

Daily Pre-Work Safety Briefing (15 min)

  • All team members present and accounted for
  • PPE inspection: hard hats, safety glasses, gloves, steel-toe boots
  • Fall protection: harnesses inspected, anchor points identified for work >2m height
  • Weather check: no work during lightning, high winds (>40 km/h), or heavy rain
  • Electrical hazard review: DC strings generate voltage in sunlight — treat as LIVE
  • Emergency contacts posted: nearest hospital, fire department, Solaris office
  • First aid kit location confirmed and accessible
  • Tool inspection: ladders, drills, crimping tools in working condition
  • Hydration plan: water accessible on-site (Panama heat index can exceed 40°C)
  • Communication: walkie-talkies or phones for roof-to-ground coordination
STOP WORK triggers

Lightning within 10km, wind gusts >50 km/h, any electrical shock incident, unsecured roof edge without guardrails, crew member showing heat exhaustion symptoms. Zero tolerance policy — report immediately to supervisor.

Charla de Seguridad Diaria Pre-Trabajo (15 min)

  • Todos los miembros del equipo presentes
  • Inspección de EPP: cascos, lentes, guantes, botas con punta de acero
  • Protección contra caídas: arneses inspeccionados, puntos de anclaje identificados
  • Verificación climática: no trabajar durante rayos, vientos fuertes o lluvia intensa
  • Revisión de riesgo eléctrico: strings DC generan voltaje con luz solar — tratar como ACTIVO
  • Contactos de emergencia publicados: hospital más cercano, bomberos, oficina Solaris
  • Botiquín de primeros auxilios verificado y accesible
  • Inspección de herramientas: escaleras, taladros, herramientas de crimpado
  • Plan de hidratación: agua accesible en sitio (índice de calor >40°C en Panamá)
  • Comunicación: radios o teléfonos para coordinación techo-suelo
Disparadores de PARO DE TRABAJO

Rayos a menos de 10km, ráfagas >50 km/h, cualquier incidente de choque eléctrico, borde de techo sin barandas, síntomas de golpe de calor. Política de tolerancia cero.

תדריך בטיחות יומי טרום-עבודה (15 דק')

  • כל חברי הצוות נוכחים ומדווחים
  • בדיקת ציוד מגן: קסדות, משקפי מגן, כפפות, נעלי בטיחות
  • הגנה מנפילה: רתמות נבדקו, נקודות עיגון זוהו לעבודה מעל 2 מ'
  • בדיקת מזג אוויר: אין עבודה בברקים, רוחות חזקות (>40 קמ"ש), או גשם כבד
  • סקירת סכנות חשמל: מחרוזות DC מייצרות מתח באור שמש — טפל כ-חי
  • אנשי קשר חירום תלויים: בית חולים קרוב, כיבוי אש, משרד סולאריס
  • מיקום ערכת עזרה ראשונה מאושר ונגיש

Common Issues in Panama Problemas Comunes en Panamá בעיות נפוצות בפנמה

Roof Penetrations on Metal Roofs

Most commercial and many residential buildings in Panama have corrugated metal roofs (zinc or Galvalume). Key considerations:

  • Sealing: Every penetration must use EPDM rubber washers + polyurethane sealant (Sikaflex 221)
  • Flashing: Use metal flashing boots over any pipe or conduit penetrations
  • Thermal expansion: Metal roofs expand/contract significantly — use slotted mounting holes
  • Rust prevention: Touch up any scratches or drill holes with zinc-rich primer immediately
  • Preferred method: Clamp-on mounting (S-5! or K2 S-Dome) to avoid penetrations entirely when possible

Tropical Storm Preparation

  • Wind load: Design for 160 km/h minimum (Panama is outside main hurricane belt but gets tropical storms)
  • Tilt angle: Lower tilt angles (10-15°) reduce wind uplift — preferred in Azuero Peninsula
  • End clamps: Use reinforced end clamps on perimeter panels
  • Wire management: All cables must be secured with UV-rated zip ties every 60cm — loose cables become wind sails
  • Seasonal timing: Avoid scheduling installations during peak rainy season (Oct-Nov) when possible
  • Storm protocol: If tropical storm warning issued, verify all installations completed in last 30 days — conduct emergency tightening inspection

Other Panama-Specific Challenges

  • Heat: Schedule heavy roof work for early morning. No work 12-1 PM during dry season heat peaks
  • Afternoon rain: During rainy season (May-Dec), plan electrical work for mornings — rain typically starts 1-3 PM
  • Wildlife: Check for bird nests and wasp nests before mounting. Iguana damage to cables is rare but documented
  • Accessibility: Many rural Azuero sites have limited road access — confirm truck clearance before delivery
  • Power outages: Have portable generator on site for power tools. Panama grid can be unreliable in rural areas

Penetraciones en Techos Metálicos

La mayoría de edificios comerciales y muchos residenciales en Panamá tienen techos de metal corrugado. Consideraciones clave:

  • Sellado: Cada penetración debe usar arandelas EPDM + sellador poliuretano (Sikaflex 221)
  • Flashing: Usar botas metálicas sobre cualquier penetración de tubo o conducto
  • Expansión térmica: Techos metálicos se expanden/contraen — usar agujeros de montaje ranurados
  • Prevención de óxido: Retocar raspaduras con imprimante rico en zinc inmediatamente
  • Método preferido: Montaje por abrazadera (S-5! o K2 S-Dome) para evitar penetraciones

Preparación para Tormentas Tropicales

  • Carga de viento: Diseñar para mínimo 160 km/h
  • Ángulo de inclinación: Ángulos bajos (10-15°) reducen levantamiento por viento
  • Abrazaderas finales: Usar reforzadas en paneles perimetrales
  • Gestión de cables: Asegurar con amarres UV cada 60cm
  • Temporada: Evitar instalaciones en pico lluvioso (Oct-Nov) cuando sea posible
  • Protocolo de tormenta: Si hay alerta, verificar instalaciones últimos 30 días

Otros Desafíos Específicos de Panamá

  • Calor: Programar trabajo pesado de techo temprano. No trabajar 12-1 PM en temporada seca
  • Lluvia vespertina: En temporada lluviosa, planificar trabajo eléctrico por las mañanas
  • Fauna: Verificar nidos de aves y avispas antes del montaje
  • Accesibilidad: Muchos sitios rurales en Azuero tienen acceso limitado — confirmar paso de camión
  • Cortes de luz: Tener generador portátil para herramientas eléctricas

חדירות בגגות מתכת

רוב המבנים המסחריים ורבים מהביתיים בפנמה בנויים עם גגות מתכת גלית. שיקולים עיקריים:

  • איטום: כל חדירה חייבת לכלול אטמי EPDM + חומר איטום פוליאורתן (Sikaflex 221)
  • פלשינג: להשתמש בכיסויי מתכת על כל חדירת צנרת
  • התפשטות תרמית: גגות מתכת מתרחבים/מתכווצים — להשתמש בחורי הרכבה מחורצים
  • מניעת חלודה: לתקן שריטות עם פריימר עשיר באבץ מיד
  • שיטה מועדפת: הרכבה בלחיצה (S-5! או K2 S-Dome) כדי להימנע מחדירות

הכנה לסופות טרופיות

  • עומס רוח: תכנון למינימום 160 קמ"ש
  • זווית הטיה: זוויות נמוכות (10-15°) מפחיתות עילוי רוח
  • מלחצי קצה: להשתמש במחוזקים בפאנלים היקפיים
  • ניהול כבלים: לאבטח עם אזיקוני UV כל 60 ס"מ
  • עונתיות: להימנע מהתקנות בשיא הגשם (אוק-נוב) כשאפשר
  • פרוטוקול סופה: בהתרעה, לבדוק את כל ההתקנות מ-30 יום אחרונים

אתגרים נוספים ספציפיים לפנמה

  • חום: לתזמן עבודה כבדה על הגג לבוקר. אין עבודה 12-1 בעונה היבשה
  • גשם אחה"צ: בעונת הגשמים, לתכנן עבודה חשמלית לבקרים
  • חיות בר: לבדוק קיני ציפורים וצרעות לפני הרכבה
  • נגישות: אתרים כפריים רבים עם גישה מוגבלת — לאשר מעבר משאית
  • הפסקות חשמל: גנרטור נייד באתר לכלים חשמליים

Client Communication During Installation Comunicación con el Cliente Durante Instalación תקשורת עם לקוח במהלך התקנה

Daily WhatsApp Updates

Send progress updates to the client via WhatsApp at the end of each installation day. Format:

  • Subject: "Solaris Update — [Project Name] — Day [X]"
  • Content: 2-3 sentence summary of what was completed
  • Photos: 3-5 key progress photos (see documentation requirements above)
  • Next: What will happen tomorrow
  • Issues: Any concerns or changes (communicate immediately, don't wait for end of day)
  • Tone: Professional but friendly. Client should feel informed and confident

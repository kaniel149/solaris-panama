# Solaris Panama — Sales Academy Design

**Date:** 2026-07-02 · **Status:** Approved by Kaniel

## Goal
Training academy for **new local Panamanian salespeople** (Spanish, no solar background) covering the **full sales cycle**: first contact (cold + warm) → discovery → site visit → proposal → objections → close → follow-up. Salesperson owns the deal end-to-end; Kaniel only approves pricing.

## Format
Modular markdown in `sales/academy/`. All trainee content in **Spanish (Panama, "usted")**. Manager guide in Hebrew. Exportable to PDF later.

## Structure (journey-based, script cards per stage)
| File | Module |
|---|---|
| README.md | Index + how to use + 30-day onboarding path |
| 00-guia-del-gerente.md | Manager guide (Hebrew) — how to run the training |
| 01-fundamentos.md | Solar basics, Ley 417, Panama electric bill, ROI concept |
| 02-producto-y-oferta.md | 3 payment options, pricing logic, fit per segment |
| 03-lead-caliente.md | Warm LP leads — speed-to-lead, 60-sec script, WhatsApp+phone |
| 04-lead-frio.md | Cold B2B — hotels/supermarkets/fincas scripts, gatekeepers |
| 05-descubrimiento.md | Discovery questions, qualification, booking the visit |
| 06-la-visita.md | Site visit, trust building, ROI demo, presentation |
| 07-objeciones.md | 12+ objections (extends existing 7), LAER technique |
| 08-cierre.md | Closing techniques, urgency (Ley 417), signing |
| 09-seguimiento-crm.md | Follow-up, lost deals, referrals, CRM hygiene |
| 10-certificacion.md | 8 role-play scenarios + quizzes + certification criteria |

## Module template
Objetivos → short theory (beginner language) → word-for-word scripts → common mistakes → role-play exercise → checklist → quiz (answers at end).
